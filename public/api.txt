
接口域名  http://cg.aoranjianzhong.com/api/api/
接口方式  post
接口返回  code 0=失败 1=成功 100=未登录 101=登录失效   -1=token没传 -2=token错误 -3=token不是该用户的  999=token过期
         data   返回数据
         message    对应的信心描述
接口要求  token验证  除了[wxlogin] 几个接口外
         其余接口都需要传递 accessToken（getToken接口返回的accessToken值） 参数进行token验证
         放在header请求头里面传递


接口名称    微信授权登录
接口路径    wxLogin
接口参数    nickname            昵称
            openid              opneid
            unionid
            headimgurl          头像
返回参数     成功后自动登录 返回用户信息

接口名称    获取token
接口路径    getToken
接口参数
返回参数

返回参数    src             图片地址 相对

接口名称    文件上传
接口路径    uploadFile
接口参数    file            文件对象
           type           上传对象 1-头像 2-商品图片
返回参数    src             文件地址 相对

接口路径    messageEdit

            avatar              头像地址

接口名称    房间挑战-房间类型
接口路径    roomType
接口参数
返回参数    type                1-保底 2-普通
            minMoney            最低金额 单位元
            maxMoney            最高金额
            rule                规则说明
            percent             失败金百分比奖励

接口名称    房间挑战-创建房间
接口路径    roomCreate
接口参数    type                1-保底 2-普通
            number              人数限制  0-不限制 默认是0
            name                房间名称
            desc                房间描述
            money               报名金额
            beginDate           开始时间  2020-09-02
            day                 活动周期 单位天
            signBegin           首次签到开始时间 12:00
            signEnd              首次签到结束是阿金 13:00
            signNum             签到次数  1或者2 默认1
            secondBegin         二次签到开始时间  signNum为2必传  15:00
            secondEnd           二次签到结束时间  signNum为2时必传 16:00
返回参数

接口名称    房间挑战-挑战列表
接口路径    roomList
接口参数    type            默认0 0-所有 1-保底 2-普通
            page            页数 默认1
            pageSize        每页数量 默认10
返回参数
